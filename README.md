# Лаб.2. Буферы, процедурное окрашивание, анимация

1. Разместите в одном окне вывода одновременно
   1. Монотонно окрашенный правильный пятиугольник.
   2. Монотонно окрашенный кубик. (см. Лекция 2 — только кубик нужен статический — динамика пока не нужна, просто поверните один раз, чтобы кубик был под углом)
   3. Квадрат в полоску. (см. Лекция 2 — полоска по образцу клетки для кубика)

Результаты должны выглядеть примерно так, только в одном окне:

2. Вывести в одном окне 3 кубика с окрашиванием, сгенерированным процедурно: квадратики (пример из лекции), диагональная штриховка, горизонтальная полоска.

3. Вывести кубик с гранями, окрашенными в разные цвета (для передачи использовать вершинные и индексные буферы)

4. Реализовать управление (перемещение, масштабирование, поворот) с клавиатуры
